## PopWindowOptions

| 属性名 | 描述 | 类型 | 默认值 |
| ---- | ---- | ---- | ---- |
| delta | 返回的页面数, 如果 delta 大于现有页面数, 则返回到首页 | `number` | `1` |
| data | 对象将会被即将露出的页面通过 onResume 事件接收 | `object` | - |
